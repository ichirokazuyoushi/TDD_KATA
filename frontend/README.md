# Sweet Shop Frontend

React frontend application for the Sweet Shop Management System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- User authentication (Login/Register)
- Sweet shop dashboard
- Search and filter sweets
- Purchase functionality
- Admin panel for managing sweets (CRUD operations)
- Responsive design

## Tech Stack

- React 18 with TypeScript
- Vite
- React Router
- Axios
- Modern CSS
